# Assigning Values To The Variables

```python

name = "Sangamesh"
height = 5.8
weight = 51
is_student = True

```

## Here,

name, height, weight, is_student are the 
### variables.

"sangamesh", 5.8, 51, True Are the ### values.which are assigned to the variables.  


```python

print(name)
#O/P: sangamesh

print(height)
#O/P: 5.8

print(weight)
#O/P: 51

print(is_student)
#O/P: True

```

# Assigning the values to Multiple variables:

### Ex:01
 
```python
x = 10
y = 20
z = 30

print(x)
#O/P: 10

print(y)
#O/P: 20

print(z)
#O/P: 30

```

### Ex:02

```python
x, y, z = 10,20,30

print(x)
#O/P: 10

print(y)
#O/P: 20

print(z)
#O/P: 30

print(x, y, z)
#O/P: 10 20 30

```

### Ex:03

```python
x = y = z = 10 
print(x)
#O/P: 10

print(y)
#O/P: 10

print(z)
#O/P: 10

```

## Variable reassignment:
We can Assign infinite values to the same variable that is called variable reassignment.

```python
x = 10
print(x)
#O/P: 10

x = 100
print(x)
#O/P: 100

x = "sangamesh" 
print(x)
#O/P: 'sangamesh'

x = 5.8
print(x)
#O/P: 5.8

x = True
print(x)
#O/P: True

```
